# TASKILY Dashboard

TASKILY dashboard built with JavaScript — practical features for real-world use.

## Features

- Dashboard overview with stats and charts
- Task, project, team, and calendar pages
- Reusable UI components and modal system
- Light/dark appearance support

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Recharts

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open in browser:

`http://localhost:3000`  
If port 3000 is busy, Next.js uses the next available port (for example `3001`).

## Production

Build and start:

```bash
npm run build
npm run start
```
